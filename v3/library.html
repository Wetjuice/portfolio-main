The Art of Game Design
Jesse Schell · 2008
2008
READING
// notes
The most comprehensive framework for thinking about games as designed experiences. The "lens" system is a practical toolkit for evaluating any design decision. Required reading — return to it at every stage of a project.
-rw-r--r--
A Theory of Fun for Game Design
Raph Koster · 2004
2004
REFERENCE
// notes
The foundational argument that fun = learning. Games are enjoyable because the brain is wired to find pattern recognition satisfying. Understanding this changes how you think about difficulty curves and player mastery arcs.
-rw-r--r--
The Design of Everyday Things
Don Norman · 1988
1988
REFERENCE
// notes
Norman's affordances and signifiers apply directly to game UX. Players shouldn't need a tutorial to understand a well-designed interface. When an action is unclear it's a design failure, not a player failure.
-rw-r--r--
Thinking, Fast and Slow
Daniel Kahneman · 2011
2011
REFERENCE
// notes
System 1 vs System 2 thinking is directly applicable to game UI/UX. Players make most in-game decisions on autopilot — good design works with that, not against it. Cognitive load is the enemy of immersion.

Alex Jaffe · GDC 2020
2020
REFERENCE
// notes
Identifies design problems where every solution creates a new problem. The concept of "cursed problems" — constraints that make a perfect solution impossible — is essential context for directors managing stakeholder expectations.

Anthony Pecorella · GDC 2015
2015
REFERENCE
// notes
Breaks down engagement loops to their atomic components. Useful for understanding why progression systems work even when stripped of all other mechanics. Core loop clarity is the foundation of any successful live service game.

How to Explain Your Game to an Asshole
Tom Francis · GDC 2012
2012
REFERENCE
// notes
Every game director needs to pitch and re-pitch their game constantly. This talk cuts through the noise of marketing-speak and teaches you to communicate the actual experience. If you can't explain it clearly, the design isn't clear enough.
